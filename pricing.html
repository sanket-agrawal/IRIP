10 —Cost Optimization Strategies
🧠 Maximize Browser-Side AI Saves $50–100/mo
ONNX Transformers.js runs embeddings and basic classification in the browser — free. Only call Groq for narrative generation (1–3% of events). With prompt caching + dedup via Redis, LLM costs stay under $20/mo even at scale.
🗄️ Aggressive Caching Saves 80% on API calls
Set s-maxage headers correctly per source: USGS (30s), NOAA (60s), GDELT (15min), ACLED (6h). Upstash Redis warm cache reduces upstream calls by ~80%, keeping Vercel Function invocations well within free/Pro limits.
🗺️ Self-Host Map Tiles Saves $20–200/mo
Use Protomaps + a single .pmtiles file on Cloudflare R2 ($0.015/GB/mo). 120GB planet file costs ~$1.80/mo in storage vs $20–200/mo in MapTiler fees. One-time setup effort; massive long-term savings.
✈️ OpenSky over AviationStack Saves $29+/mo
OpenSky provides unlimited free ADS-B flight tracking for monitoring. Use AviationStack only if you need commercial flight status/delay data. For the risk monitoring layer, raw ADS-B positions are sufficient.
🌊 Open-Meteo over Stormglass Saves $39/mo
Open-Meteo Marine API provides unlimited wave height, swell, wind data for free. Completely replaces Stormglass ($39–199/mo) for the marine routing risk layer. Same data quality for monitoring purposes.
🚢 GDELT as ACLED substitute Saves $200–500/mo
GDELT covers 300+ event types globally, updates every 15 minutes, and is 100% free forever — no commercial licensing. For a monitoring dashboard (vs academic research), GDELT + UCDP + Polymarket covers political risk without ACLED's commercial fee.
📦 Groq Batch API 50% discount
For non-urgent LLM classification (e.g., batch-processing the overnight GDELT dump), use Groq's Batch API for 50% off. Only use real-time API for time-sensitive events. Prompt caching also reduces cost for repetitive system prompts.
🔐 Defer Enterprise Auth Saves $100–225/mo
Clerk's free tier covers 10,000 MAU with no MFA add-on required at MVP stage. Defer the Enhanced Auth add-on (+$100/mo) until enterprise clients demand SSO/SAML. Use Clerk's built-in TOTP (free) for basic MFA.
Budget Risk Factors
⚠️ Vercel Overage Spikes If a major cat event causes traffic spikes, Edge Request overages ($2/M) can accumulate fast. Set Vercel's spend management budget cap at $50–100/mo to auto-pause. Fix: aggressive CDN caching reduces requests by 90%.
⚠️ ACLED Commercial Licensing The moment you charge clients, ACLED non-commercial terms no longer apply. Budget $200–500/mo for commercial license or replace entirely with GDELT (free forever, no commercial restriction).
⚠️ LLM Token Runaway If every event triggers an LLM call without deduplication, costs explode. Solution: in-memory dedup + Redis cross-user dedup ensures the same headline is only classified once. Set hard Groq monthly budget cap.
⚠️ Railway Memory Overflow WebSocket relay keeping many concurrent connections can spike memory. Railway bills by utilization — monitor usage dashboard. Set auto-restart on OOM. Hobby plan caps at 8GB RAM max.
⚠️ MapTiler Tile Explosion Each user panning/zooming the map triggers tile requests. 10 users × 500 tile requests/session = 5K tiles per session. Consider self-hosting with Protomaps from day one to eliminate this risk entirely.
⚠️ MarineTraffic Free Limit 50 calls/day free is very limiting for AIS data. The 3-tier cache makes this workable (cache AIS positions for 30s). Growth: upgrade to Basic API ($49/mo) or use USCG NAIS (free) for US waters.
