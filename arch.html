System Philosophy
IRIP follows WorldMonitor's three-tier model: (1) External APIs → ingested via Edge Functions with 3-tier cache, (2) AI engine computes risk scores in-browser using embedded ONNX models + LLM fallback, (3) Map renderer overlays scored events as interactive layers. All 5 insurance lines share the same core infrastructure; domain logic is modular and additive.

Intelligence Engine
Risk Scoring Pipeline
01
Ingest
Raw API responses normalized to RiskEvent schema. Deduplication via SHA-256 event fingerprint.
02
Embed
ONNX all-MiniLM-L6-v2 in Web Worker produces 384-dim vector per headline. Stored in IndexedDB.
03
Classify
Keyword fast-path → insurance line tag (P&C, Marine, etc). Groq LLM override for edge cases.
04
Baseline
Welford's algorithm builds rolling 90-day mean/variance per event-type × region × weekday × month.
05
Score
Z-score vs baseline → σ 1.5 = Elevated, σ 2.0 = High, σ 3.0 = Critical. Composite per insurance line.
06
Render
Scored events pushed to map layers via Event Bus. Severity drives color, size, pulse animation.

🌪️ Nat Cat Risk Index
NatCatRisk = ( EQ_Score × 0.30 + // USGS magnitude/depth Fire_Score × 0.25 + // FIRMS radiative power Flood_Score × 0.20 + // NFIP + GDACS Storm_Score × 0.15 + // NOAA alerts Volcano_Score × 0.10 ) × Exposure_Multiplier
USGS M-value FIRMS FRP GDACS alert level Population density NFIP claims history
✈️ Aviation Risk Index
AviaRisk = ( AirspaceClosure × 0.35 + // NOTAMs GPSJam × 0.25 + // GPS interference ConflictOverflight × 0.20 + // ACLED near routes WeatherSeverity × 0.15 + // NOAA SIGMETs IncidentRate × 0.05 // ASN 90-day rolling ) × Route_Exposure
NOTAM type GPSJam density Active conflict radius SIGMET coverage
⚔️ Political Risk Index
PolRisk = ( ConflictIntensity × 0.35 + // ACLED fatalities/events GovInstability × 0.25 + // GDELT tone score SanctionsExposure × 0.20 + // OFAC SDN hits ProtestScale × 0.10 + // GDELT crowd events PredictiveMarket × 0.10 // Polymarket odds ) × Country_Weight
ACLED fatality rate GDELT avg tone OFAC exposure Polymarket probability
🚢 Marine Risk Index
MarineRisk = ( PiracyZone × 0.30 + // ICC-IMB incidents ChokeDisruption × 0.25 + // AIS congestion index WeatherRouting × 0.20 + // Stormglass wave/wind ConflictAdj × 0.15 + // ACLED coastal CargoType × 0.10 // Hazmat multiplier ) × Voyage_Exposure
ICC-IMB zone code AIS chokepoint delay Wave height (m) Cargo value tier
🛡️ Cyber Risk Index
CyberRisk = ( ThreatActivity × 0.30 + // OTX pulse volume RansomwareActive × 0.25 + // ransomware.live KEV_Exposure × 0.20 + // CISA KEV match OutageSignal × 0.15 + // Cloudflare Radar C2Activity × 0.10 // Feodo Tracker ) × Sector_Multiplier
OTX pulse count Active RaaS groups CVSS score BGP anomaly
📊 Composite Exposure Score
ExposureScore = MAX( NatCatRisk, AviaRisk, PolRisk, MarineRisk, CyberRisk ) × SQRT( SUM(all_active_scores) / N ) // Max dominant + RMS of all active lines
Per-line scores Temporal decay Correlation matrix Portfolio exposure
